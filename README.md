# Inventory-System
Inventory management software is a computer-based system for
tracking inventory levels, orders, sales and deliveries. It is a tool for organizing
inventory data that before was generally stored in hard-copy form or in
spreadsheets.This project is aimed at developing a general Inventory Management System.
This system can be used to store the details of the inventory, update the
inventory based on the sale details, produce receipts for sales, generate sales
and inventory reports periodically etc.
